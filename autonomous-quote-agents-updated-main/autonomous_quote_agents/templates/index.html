Problem Statement

Auto insurance carriers generate thousands of quotes daily through Exclusive Agents (EA) and Independent Agents (IA) across multiple regions. Yet only 1 in 5 quotes ever converts to a bound policy. This system implements a 4-agent AI pipeline that autonomously processes quotes end-to-end — profiling risk, predicting conversion, negotiating premiums, and routing decisions — without requiring a human in the loop for every step.

Multi-Agent Orchestration ML Core (RF + GBM) Chain-of-Thought Explainability Human Escalation Protocol
4-Agent Architecture
Each agent consumes structured outputs from the previous agent and passes enriched signals downstream. The pipeline is fully autonomous — a human only sees quotes that reach the Escalate bucket.

RandomForest · 200 trees · class_weight=balanced
FULLY AUTO
Ingests raw quote data and computes a real-time risk tier (Low / Medium / High) using driver history, age, experience, and usage patterns. Drives all downstream agent decisions.

Decision Thresholds
✅ Auto-Approve
  • Bind probability ≥ 65%
  • Risk score ≤ 40
  • Agent 1 confidence ≥ 75%
  • No premium blocker detected
🔵 Agent Follow-Up
  • Moderate bind probability (20–65%)
  • Premium blocker may be present
  • Coverage downgrade recommended
  • Any other middle-ground signal
🔴 Escalate to Underwriter
  • Risk score ≥ 70
  • Agent 1 confidence ≤ 50%
  • Bind prob ≤ 20% with High risk tier
  • Any uncertain edge case

      Explainability Framework

      Every agent decision in this pipeline is explainable. The system uses three complementary techniques to justify routing decisions to underwriters and regulators.

      Chain-of-Thought Traces

      Every routing decision emits a step-by-step log of agent inputs, threshold comparisons, and the final routing rationale. Visible in the Analyze Quote tab.

      Feature Attribution (LIME-style)

      Agent 2 reports the top drivers of its bind probability score (re-quote behavior, risk tier, coverage preference, salary level) in plain English for every prediction.

      Domain-Rule Audit Trail

      Agent 3 (Hybrid) documents every rule it fires — affordability threshold, vehicle cost ratio, re-quote discount — so underwriters can audit premium adjustments.

      Human Escalation Protocol

      The escalation protocol is designed to minimize human workload while maintaining accuracy. A quote reaches a human underwriter only when one or more of these conditions are triggered:

      Risk Score ≥ 70
      Agent 1 assigns a risk score above 70/100, indicating high prior incidents or dangerous driver profile. Auto-approval not safe.
      Agent 1 Confidence ≤ 50%
      When the Random Forest produces low-confidence risk tier assignment, the case is inherently ambiguous and requires human judgment.
      Bind Prob ≤ 20% + High Risk Tier
      A high-risk customer with very low conversion probability represents a special case — writing or declining this policy needs underwriter expertise.
      Regional & Channel Intelligence

      The pipeline encodes Region (A–H) and Agent Type (EA/IA) as features in Agent 2, allowing the Conversion Predictor to learn regional conversion patterns. EA agents historically show higher bind rates than IA agents in the training data — this signal is captured and used to calibrate bind probability scores per channel.
